# 章节首言
2025年1月，我购入一块200多元的华擎j3455itx，配合现有的硬盘、机箱和电源组建了我的第一台NAS，装上了群晖开心版，
陪伴了我半年的时间。接着在2025年暑期，我更换到了国产飞牛OS系统，低配下，飞牛os体验比群晖开心版要好很多。
随着我NAS使用的深入，对docker、虚拟机的需求越来越强烈，2025年10月，我升级到了更高配的i3-8100cpu+16GB内存。
