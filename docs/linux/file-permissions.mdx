# 文件权限管理

在 Linux 中，每个文件和目录都有权限设置，用于控制谁可以读取、写入或执行它。

## 查看权限

使用 `ls -l` 命令可以查看文件权限。

```bash
ls -l my-file.txt
```

输出示例：
`-rwxrw-r-- 1 user group 4096 Jan 1 12:00 my-file.txt`

- 第一个字符 (`-`) 表示文件类型（`-` 为文件, `d` 为目录）。
- 后面九个字符分为三组，分别代表 **所有者 (user)**、**所属组 (group)** 和 **其他用户 (others)** 的权限。
- `r` (读取), `w` (写入), `x` (执行)。

## 修改权限 `chmod`

`chmod` 命令用于修改文件或目录的权限。

### 符号模式

```bash
# 为所有者添加执行权限
chmod u+x my-script.sh

# 为所属组和其他用户移除写权限
chmod go-w my-file.txt

# 为所有用户设置读写权限
chmod a=rw my-file.txt
```

### 数字模式

权限可以用数字表示：`r=4`, `w=2`, `x=1`。

- `rwx` = 4 + 2 + 1 = 7
- `rw-` = 4 + 2 + 0 = 6
- `r-x` = 4 + 0 + 1 = 5

```bash
# 设置权限为 rwxr-xr-x (所有者读写执行，组和其他用户读和执行)
chmod 755 my-script.sh

# 设置权限为 rw-rw-r-- (所有者和组读写，其他用户只读)
chmod 664 my-file.txt
```
