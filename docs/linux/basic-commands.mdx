# Linux 基础命令

本教程介绍一些最常用的 Linux 命令，用于文件系统导航和操作。

## 文件和目录操作

### `ls` - 列出目录内容

```bash
# 列出当前目录的文件和文件夹
ls

# 列出详细信息
ls -l

# 列出所有文件（包括隐藏文件）
ls -a
```

### `cd` - 切换目录

```bash
# 进入 home 目录
cd ~

# 进入上一级目录
cd ..

# 进入名为 documents 的子目录
cd documents
```

### `pwd` - 显示当前工作目录

```bash
pwd
```

### `mkdir` - 创建目录

```bash
mkdir new-directory
```

### `touch` - 创建空文件

```bash
touch new-file.txt
```

### `cp` - 复制文件或目录

```bash
cp source.txt destination.txt
```

### `mv` - 移动或重命名文件或目录

```bash
# 重命名
mv old-name.txt new-name.txt

# 移动
mv file.txt /path/to/destination/
```

### `rm` - 删除文件或目录

```bash
# 删除文件
rm file.txt

# 递归删除目录及其内容
rm -r directory-name
```
