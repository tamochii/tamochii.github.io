# Linux 简介

Linux 是一种开源的、免费的操作系统内核。它由林纳斯·托瓦兹（Linus Torvalds）于1991年首次发布。基于 Linux 内核的完整操作系统被称为 Linux 发行版，例如 Ubuntu、CentOS、Debian 等。

## 为什么学习 Linux？

- **开源免费**: 你可以自由地使用、修改和分发它。
- **稳定和安全**: Linux 以其稳定性和强大的安全特性而闻名，是服务器操作系统的首选。
- **强大的命令行**: 命令行界面（CLI）提供了高效和灵活的方式来管理系统。
- **广泛应用**: 从服务器、嵌入式设备到桌面计算机，Linux 无处不在。

## 文件系统结构

Linux 遵循文件系统层次结构标准（FHS）。一些重要的目录包括：

- `/`: 根目录，所有文件和目录的起点。
- `/bin`: 存放基本的用户命令。
- `/etc`: 存放系统配置文件。
- `/home`: 用户的主目录。
- `/var`: 存放经常变化的文件，如日志文件。
- `/tmp`: 存放临时文件。
