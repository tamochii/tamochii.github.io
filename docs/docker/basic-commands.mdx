# Docker 基础命令

本教程介绍一些最常用的 Docker 命令。

## 镜像相关命令

### `docker pull`

从 Docker Hub 拉取一个镜像。

```bash
docker pull ubuntu:20.04
```

### `docker images`

列出本地的所有镜像。

```bash
docker images
```

### `docker rmi`

删除一个或多个镜像。

```bash
docker rmi ubuntu:20.04
```

## 容器相关命令

### `docker run`

创建一个新的容器并运行。

```bash
# 在后台运行一个 nginx 容器
docker run -d -p 8080:80 --name my-nginx nginx
```

### `docker ps`

列出正在运行的容器。

```bash
# 列出所有容器（包括已停止的）
docker ps -a
```

### `docker stop` / `docker start`

停止或启动一个容器。

```bash
docker stop my-nginx
docker start my-nginx
```

### `docker rm`

删除一个或多个容器。

```bash
docker rm my-nginx
```
