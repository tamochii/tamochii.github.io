# Docker 简介

Docker 是一个开源的应用容器引擎，让开发者可以打包他们的应用以及依赖包到一个可移植的容器中，然后发布到任何流行的 Linux 机器或 Windows 机器上，也可以实现虚拟化。

## 为什么使用 Docker？

- **环境一致性**: 确保开发、测试和生产环境的一致性。
- **快速部署**: 容器是轻量级的，启动速度快。
- **隔离性**: 每个容器都运行在独立的环境中。
- **可移植性**: 一次构建，随处运行。

## 核心概念

- **镜像 (Image)**: 一个只读的模板，包含创建 Docker 容器的说明。
- **容器 (Container)**: 镜像的运行实例。
- **仓库 (Repository)**: 集中存放镜像文件的地方。
