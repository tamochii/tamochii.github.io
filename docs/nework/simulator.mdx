# 网络模拟器
现在高校常用的网络都是华为的老 eNSP，但 eNSP 已经很久没有更新了，兼容性堪忧。  
所以本文将推荐几款好用的网络模拟器和安装教程，供大家选择使用。

## Cisco Packet Tracer
Cisco Packet Tracer 是思科官方推出的一款网络模拟器。

## EVE-NG
EVE-NG 是一款功能强大的网络模拟器，支持多种网络设备的模拟和虚拟化。

## PNETLab
PNETLab 基于 EVE-NG 社区版源码开发，相当于在 EVE-NG 的框架上增加了更多功能和改进

## H3C Cloud Lab
H3C Cloud Lab 是华三官方推出的网络实验平台，支持 H3C 设备的模拟和虚拟化。
