# 网络模拟器
现在高校常用的网络都是华为的老 eNSP，但 eNSP 已经很久没有更新了，兼容性堪忧。
所以本文将推荐几款好用的网络模拟器/仿真器和安装教程，供大家选择使用。
下文仅仅是对其的结束，不包含安装说明和使用说明，具体安装和使用可以自行 google，本人常用 CPT 和 PNETLab，后续会补充相关文章。

## Cisco Packet Tracer (CPT)
> 零基础新手的“入坑”首选，轻量级，不折腾。

这也是思科网院的官方工具。严格来说，它属于**仿真器**而不是模拟器。它不运行真实的操作系统，只是模拟了设备的逻辑。

* **为什么选它：**
    * **极度省心：** 安装包就几百兆，双击装好就能用。不需要像别的软件那样去到处找镜像文件（IOS）。
    * **电脑不发烫：** 哪怕你是那种轻薄本，跑个复杂的拓扑也毫无压力，因为它不吃 CPU 和内存。
    * **可视化无敌：** 它有个物理视图，能看见机柜、线缆，甚至数据包的流动都是动画展示的，对理解“数据包怎么走”非常有帮助。
* **缺点：**
    * 既然是仿真的，很多高级命令就没有。你想做点复杂的 MPLS 或者 VXLAN 实验？它不行。
    * 只支持思科设备。

## EVE-NG (Emulated Virtual Environment)
> 行业标准，网工界的“瑞士军刀”，想变强必玩。

现在的中高级网工，甚至去大厂面试，EVE-NG 基本上是标配技能。它本身是一个基于 Ubuntu 的虚拟机，你把它装在 VMware 里，然后通过浏览器去操作。

* **为什么选它：**
    * **通吃所有厂商：** 它是真的在跑设备镜像。无论是思科、华为、华三、Juniper，还是 Linux 服务器、Windows，甚至防火墙，只要你有镜像，它都能跑。
    * **环境真实：** 因为跑的是真实系统，你在里面敲的命令、看到的现象，和真机是一模一样的。
    * **B/S 架构：** 服务端搭好后，你可以躺在床上用平板浏览器敲实验，非常优雅。
* **劝退点：**
    * **门槛高：** 安装过程涉及 Linux 操作，导入镜像如果不看教程容易懵。
    * **资源黑洞：** 极其吃内存！如果你想跑个稍微大点的拓扑，电脑 16G 内存起步，32G 才算舒服。

## PNETLab
> EVE-NG 的“魔改增强版”，懒人福音。

PNETLab 其实是基于 EVE-NG 社区版源码二次开发的，可以看作ubuntu和debian的关系吧。

* **为什么选它：**
    * **功能全开：** 原版 EVE 里一些需要收费的高级功能（比如实验室管理、Docker 支持），PNETLab 直接免费给你用。
    * **镜像好找：** 它自带一个类似“应用商店”的模式（虽然有时候需要魔法上网），下载镜像比 EVE 方便太多了，不用满世界找资源。
    * **界面好看：** UI 做了美化，操作逻辑也更顺手。

## H3C Cloud Lab (HCL)
> 华三考证专用，更像是一款持久维护的eNSP，如果习惯 eNSP 首选HCL。

如果你学校教的是华三体系，或者你要考 H3C 的证，那 HCL 是绕不开的。

* **现状：**
    * 它是华三官方出的，所以对 Comware V7 平台的模拟是最完美的。
    * 虽然它也依赖 VirtualBox，但兼容性比老 eNSP 稍微好那么一点点（建议配合 VirtualBox 6.0 使用，别用最新的）。
    * 界面虽然有点复古，但该有的功能都有。

## 总结

| 模拟器/仿真器 | ✅ 优点 | ❌ 缺点 | 推荐指数 |
| :--- | :--- | :--- | :--- |
| **Cisco Packet Tracer** | **零门槛**：双击安装即用，老爷机也能跑。  **可视化**：能看见数据包动画，原理神器。  **省心**：无需寻找复杂的镜像文件。 | **是假的**：仿真而非模拟，命令支持不全。   **局限性**：仅支持思科设备。 | ⭐⭐⭐⭐⭐ |
| **EVE-NG** | **万能**：思科、华为、Linux、防火墙全支持。  **真实**：运行真实镜像，环境等同现网。  **优雅**：B/S架构，浏览器直接操作。 | **折腾**：需虚拟机、Linux基础，导镜像麻烦。   **吃配置**：16G内存起步，电脑风扇狂转。 | ⭐⭐⭐⭐ |
| **PNETLab** |  **功能全**：免费解锁 EVE 收费版功能。  **方便**：自带镜像商店（需魔法），下载容易。  **美观**：UI 界面优化，交互更好。 | **稳定性**：魔改太多，偶有小 Bug。  **社区**：技术支持不如 EVE 原版硬核。 | ⭐⭐⭐⭐ |
| **HCL** | **原味**：官方出品，Comware V7 命令完美支持。  **刚需**：考 H3C 证书躲不掉的工具。 | **兼容差**：依赖 VirtualBox，Win10/11 易报错。  **体验差**：界面老旧，容易卡顿。 | ⭐⭐⭐ |
| **eNSP** | **高校支持**：大部分高校都在用。 | **BUG**：年久失修，BUG数不尽。 | ⭐ |

**最后一句话总结：**  
新手入门选 CPT，懂linux折腾选 EVE/PNETLab，考 H3C 证书选 HCL，别选 eNSP。
