## 安装 Node.js 和 npm

访问 [Node.js 官方网站](https://nodejs.org/zh-cn/download) 下载适用于自己的版本，我这里选择获得适用于 Linux 且使用 nvm 和 npm 的 Node.js v24.11.1(LTS)。  
下面是我对应版本的安装命令：

```shell
# 下载并安装 nvm：
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash

# 代替重启 shell
\. "$HOME/.nvm/nvm.sh"

# 下载并安装 Node.js：
nvm install 24

# 验证 Node.js 版本：
node -v # Should print "v24.11.1".

# 验证 npm 版本：
npm -v # Should print "11.6.2".
```

## 用 Vite 创建 React 项目

Vite 是一个现代前端构建工具，支持多种框架，包括 React。

```shell
# 创建（JS）
npm create vite@latest my-react-app -- --template react
# 或创建（TS）
npm create vite@latest my-react-app -- --template react-ts
# 进入项目目录
cd my-react-app
npm install        # 或 pnpm install / yarn
npm run dev        # 启动开发服务器
# 如需指定端口：
# npm run dev -- --port 3000
```
